Tarea:

  1. ¿Qué es un Protocolos Http?

    Http (Protocolo de transferencia de hipertexto) es un protocolo que transmite u envia las solicitudes o pedidos de datos del cliente a los servidores de web, para que estos le den una respuesta

  2. ¿Qué es un Query String?

    "Cadenas de consultas", es la forma de enviar datos a las apps web utilizando una URL mayormente con el metodo Get, asi podemos acceder a portales o paginas web con URL sencillas para los navegadores

  3. ¿Qué es un Verbos HTTP?

    Son pedidos o solicitudes para hacer uso de un recurso, que lo envia el cliente al servidor web, son 5 verbos HTTP (Get, post, Put, delete...)

  4. ¿Qué es un DNS?

    "Sistema de nombres de dominios", es un servicio de internet que permite traducir los nombres de dominios (ejmp: www.mibanco.com) a direcciones IP (ejmp: 129.12.9), esto hace posible que navegues por internet

  5. ¿Qué es un CDN?

    "Red de distribucion de contenido", es un grupo de servidores distribuidos en diferentes zonas, para acelerar la entrega de datos web a los usuarios, los datos son guardados en caché, asi se podra acceder a ellos mediante paginas web
