PREGUNTAS:

Comenzamos por contestar las siguientes interrogantes:

¿Qué es un Protocolos HTTP?

HTTP (Protocolo de transferencia de hipertexto), es un protocolo que transmite o envia las solicitudes o pedidos de datos del cliente a los servidores de web, para que estos le den una respuesta.

¿Qué es un Query String?

(Cadenas de consultas), es la forma de enviar datos a las Apps web utilizando una URL mayormente con el metodo GET, asi podemos acceder a portales o paginas web con URLS sencillos para los navegadores.

¿Qué es un Verbos HTTP?

Son pedidos o solicitudes para hacer uso de un recuso, que envia el cliente al servidor web, los verbos son los sgts:

-Get -Post

-Put -Delete

¿Qué es un DNS?

DNS "Sistema de nombres de dominio" es un servicio de internet que permite traducir los nombres de dominios(www.banco.com) a direcciones IP(129.12.9), esto hace posible que naveguemos por internet.

¿Qué es un CDN?

"Red de distribucion de contenido", es un grupo de servidores distribuidos en diferentes zonas, para acelerar la entrega de datos web a los usuarios, los datos son almacenados en caché, asi se podrá acceder a ellos mediante paginas.
